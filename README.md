# SmartDok Sunshine
UI component library for SmartDok.

## Project setup
```
yarn install
```

### Compiles library and minifies for production
```
yarn run build
```

### Lints and fixes files
```
yarn run lint
```

### Runs tests
```
yarn test:unit
```

### VSCode setup

Installed plugins:

- ESLint
- Vetur

Relevant configuration options:

```
    "files.trimTrailingWhitespace": true,
    "eslint.validate": [
        "javascript",
        "javascriptreact",
        "typescript",
        "typescriptreact",
        "vue"
    ],
    "eslint.packageManager": "yarn"
```
